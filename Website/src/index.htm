Assistant Info

What is the Assistant?

The assistant has been built using the AIY Voice Kit from Google. It connects to the Google Assistant API and allows the user to use voice commands and ask questions.

What can I ask the Assistant?

The assistant can respond to a multitude of commands such as:

  • 'What's the weather like?
  • 'What time is it?'
  • 'Can you translate this sentence for me?'
  • 'Tell me a joke!'
  • 'When did the First World War end?'
    • ...and many more!

Am I always being recorded?

Short answer: Yes. The Google Assistant API activates after using a hotword, but device has to listen all the time to react when hetword was used.
